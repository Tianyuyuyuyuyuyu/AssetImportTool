# AssetImporter
## 功能描述:
### 各类资源导入的预处理设置
**目前支持Texture、Audio、Model、Plugin、Shader、TrueTypeFont、VideoClip、HVImageFormat、Mono、SketchUp、SpeedTree等格式资源。**
